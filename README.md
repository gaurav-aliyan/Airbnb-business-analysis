# Airbnb Business Analysis Project

## Overview

This project analyzes Airbnb listing and booking data across 373 European cities to identify the major drivers of revenue, occupancy, and vacancy.

## Objectives

* Identify factors affecting Airbnb revenue
* Analyze occupancy differences across markets
* Evaluate host-performance impact
* Detect vacancy risks in seasonal cities

## Tools Used

* Python
* Pandas
* NumPy
* Power BI 

## Key Insights

* Superhost listings achieved significantly higher occupancy than non-superhost listings
* Larger listings generated nearly 4× higher revenue than small listings
* Ratings showed weak correlation with revenue
* Pricing had minimal direct impact on occupancy
* Seasonal tourism markets showed higher vacancy risk

## Business Recommendations

* Hosts should target superhost status
* Seasonal markets require dynamic pricing
* Premium listings perform best in strong-demand cities

## Files Included

* Raw datasets
* Cleaned dataset
* Analysis notebook
* Business report

## Outcome

This project demonstrates how business analysis can identify operational and strategic opportunities in Airbnb market performance.
